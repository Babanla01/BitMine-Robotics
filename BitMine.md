## üé® Design System

- **Neutral**: 
  - Dark Gray `#2D3748`

- **Primary Font**: Inter (Google Fonts)
- **Headings**: Inter Bold (600-700)
- **Body**: Inter Regular (400)
- **Font Sizes**:
  - H1: 2.5rem (40px)
  - H2: 2rem (32px)
  - H3: 1.5rem (24px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)

### Spacing System
- **Base Unit**: 8px
- **Spacing Scale**: 4px, 8px, 16px, 24px, 32px, 48px, 64px, 96px

### Components
- **Buttons**: Rounded corners (8px), hover effects, loading states
- **Cards**: Subtle shadows, rounded corners (12px)
- **Forms**: Clean inputs with focus states, validation feedback
- **Navigation**: Sticky header, mobile hamburger menu

### Icons & Imagery
- **Icon Library**: Lucide React or Heroicons
- **Style**: Line icons for consistency
- **Imagery**: High-quality photos of students building robots
- **Graphics**: Custom illustrations for empty states
